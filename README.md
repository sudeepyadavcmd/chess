# ♟️ Chess Game

A web-based Chess Game built using HTML, CSS, and JavaScript. This project allows users to play chess in a simple and interactive interface with basic game logic and move validation.

## 📌 Features
- ♟️ Standard chessboard (8x8 grid)
- 👥 Two-player gameplay (local)
- ✅ Valid move highlighting
- 🔄 Turn-based system (White vs Black)
- 🧠 Basic chess rules implemented
- 🎨 Interactive UI with responsive design

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (DOM Manipulation)
